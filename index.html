Вивчаємо Git та GitHub

Git — це система контролю версій, яка дозволяє відслідковувати зміни у вашому коді, співпрацювати з командою та повертатися до будь-якого моменту в історії проєкту.

🌿

Гілки (Branches)

Гілки дозволяють розробляти нові функції ізольовано від основного коду, не боячись зламати готовий продукт.

📦

Коміти (Commits)

Кожен коміт — це збережений знімок вашого проєкту. Ви завжди можете повернутися до будь-якого коміту.

🔄

Pull Requests

Pull Request — це запит на злиття змін. Команда переглядає код перед тим, як він потрапить у головну гілку.
